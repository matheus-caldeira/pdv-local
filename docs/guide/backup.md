# Backup dos dados

Seus dados ficam salvos **no próprio aparelho**, dentro do navegador. Isso é
ótimo para privacidade e para funcionar sem internet, mas significa que você é o
responsável por guardá-los. O backup serve para:

- Não perder nada se o navegador for limpo ou o aparelho trocado.
- Levar seus dados para outro celular, tablet ou computador.

> Limpar os dados do navegador, desinstalar o navegador ou usar **Apagar Todos os
> Dados** remove tudo. Faça backups com frequência — de preferência ao fim de
> cada dia.

## Fazer um backup (exportar)

1. Vá em **Config** (Configurações) → seção **Exportar Dados**.
2. Toque em **Exportar Tudo (JSON)**. Um arquivo (`pdv-backup.json`) será baixado.
3. Guarde esse arquivo num lugar seguro: e-mail, nuvem (Google Drive, etc.) ou um
   pen drive.

O formato **JSON** guarda tudo junto e é o ideal para backup completo. Se quiser
abrir os dados numa planilha, use **Exportar Tudo (CSV)**, que gera arquivos
separados por tipo.

## Restaurar ou migrar para outro aparelho (importar)

1. No aparelho novo, abra o PDV Local e vá em **Config** → seção **Importar
   Dados**.
2. Em **Tipo de Dado**, escolha o que vai importar (Produtos, Pedidos, Sessões ou
   Movimentações).
3. Selecione o arquivo (JSON ou CSV) e toque em **Importar**.
4. Repita para cada tipo de dado que você queira trazer.

> A importação **adiciona** os registros do arquivo aos que já existem no
> aparelho. Para evitar duplicatas, prefira importar num aparelho sem dados ou
> apague os dados antigos antes (lembrando de ter o backup à mão).

## Boas práticas

- Faça o backup **completo (JSON)** ao fim de cada dia de venda.
- Guarde o arquivo **fora do aparelho** (nuvem ou e-mail), para o caso de perda
  ou roubo.
- Antes de usar **Apagar Todos os Dados** ou trocar de aparelho, confirme que
  você tem um backup recente.
