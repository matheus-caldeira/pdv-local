# Tela da cozinha (KDS)

A **KDS** (sigla em inglês para "tela da cozinha") mostra os pedidos organizados
por etapa de preparo, para a cozinha acompanhar o que precisa fazer e marcar o
que já avançou.

> **Importante:** a KDS só aparece quando o **Controle de Status** está **ligado**
> em [Configurações](configuracoes). Com ele desligado, a tela não fica
> disponível no menu. O caixa também precisa estar aberto.

## As etapas

Os pedidos passam por quatro etapas, nesta ordem:

1. **Aceito** — pedido acabou de entrar.
2. **Em preparo** — a cozinha está produzindo.
3. **A caminho** — saiu para entrega ou está pronto para retirada.
4. **Finalizado** — concluído.

Cada etapa é uma coluna do quadro. O número ao lado do nome mostra quantos
pedidos estão naquela etapa.

## Como mover os pedidos

Cada cartão de pedido mostra o número da comanda, o cliente, os itens e o valor.
Para mudar a etapa:

- Toque em **Avançar** para mandar o pedido para a próxima etapa.
- Toque em **Voltar** para regredir uma etapa.
- No computador, você também pode **arrastar** o cartão de uma coluna para outra.

As mudanças aparecem **na hora** em todas as telas abertas — a própria KDS, a
tela de [Pedidos](pedidos) e o [Painel do cliente](painel) — sem precisar
atualizar a página.

> No celular, use as abas na parte de baixo para alternar entre as etapas e ver
> os pedidos de cada uma.
