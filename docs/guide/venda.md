# Venda rápida

A tela **Vender** é onde você monta o pedido e recebe o pagamento. É a tela que
você mais vai usar no balcão.

> Para vender, o **caixa precisa estar aberto**. Se aparecer "Abra o caixa para
> vender", toque em **Abrir Caixa** e informe o valor inicial. Veja [Caixa](caixa).

## Montar o pedido

1. Toque num produto para adicioná-lo ao pedido. Toque de novo para somar mais
   uma unidade.
2. Use os filtros de **categoria** no topo para achar os produtos mais rápido.
3. Cada produto adicionado aparece no carrinho (à direita no computador, ou na
   barra inferior no celular). O total é calculado automaticamente.

Para ajustar um item já no carrinho:

- Use os botões **−** e **+** para mudar a quantidade.
- Toque na **lixeira** para remover o item.
- Toque no ícone de **anotação** para escrever uma observação (ex: "sem cebola",
  "bem passado").

## Produtos com adicionais

Se o produto tem grupos de adicionais (ex: ponto da carne, acompanhamentos,
tamanho), ao tocar nele abre uma janela para você escolher:

1. Selecione os itens de cada grupo com os botões **+** e **−**.
2. Grupos marcados como **Obrigatório** exigem uma escolha; sem ela, não dá para
   adicionar.
3. O preço dos adicionais entra automaticamente no total. Alguns itens podem ser
   gratuitos até certa quantidade, conforme você configurou.
4. Escreva uma observação se precisar e toque em **Adicionar**.

Saiba como montar esses grupos em [Adicionais e extras](adicionais).

## Identificar o cliente (opcional)

No carrinho você pode informar:

- **Telefone:** ao digitar, o sistema sugere clientes já cadastrados. Toque numa
  sugestão para preencher nome e endereço automaticamente.
- **Nome** e **endereço:** úteis para entrega e para o histórico do cliente.
- **Comanda / Mesa:** já vem preenchida com o próximo número da sequência. Você
  pode digitar outro valor (por exemplo, o número da mesa) se preferir.

Se o telefone for informado, o cliente é cadastrado (ou atualizado)
automaticamente. Veja [Clientes](clientes).

## Finalizar e receber

Toque em **Finalizar Venda** (ou **Pagar**, no celular). Escolha uma das três
opções:

- **Pagar agora:** o cliente paga na hora. Você escolhe a forma de pagamento
  (PIX, Crédito, Débito ou Dinheiro) e confirma. O pedido entra como **pago**.
- **Abrir comanda:** o pedido fica em aberto para pagar depois (ex: mesa que vai
  consumir mais). Não precisa escolher forma de pagamento agora.
- **Pagar na entrega:** para delivery. Você escolhe a forma de pagamento que o
  cliente vai usar na entrega. O pedido entra como **pendente**.

Depois de finalizar, o carrinho é limpo e o número da comanda avança
automaticamente, pronto para a próxima venda.

> Pedidos em aberto ou pendentes podem ser pagos depois pela tela de
> [Pedidos](pedidos).

## E a comanda? Posso imprimir?

A numeração da comanda é gerada automaticamente a cada venda. A **impressão em
impressora térmica** está em preparação — por enquanto, o botão de imprimir
apenas avisa que a impressora precisa ser configurada. Acompanhe as novidades.
